# Formulario de recopilación de datos

Esta es una aplicación diseñada para recopilar datos mediante un formulario en Streamlit usando AWS.
<img width="2448" height="1522" alt="image" src="https://github.com/user-attachments/assets/c4b996a9-78c6-4f46-b632-063d2d624aab" />


## Instalación

1. Clona este repositorio en tu máquina local.
2. Abre una terminal y navega hasta la carpeta del proyecto.
3. Ejecuta el comando `streamlit run app.py` para instalar las dependencias.
4. Configura las variables de entorno necesarias para la conexión con tu proveedor de servicios de correo electrónico.

## Uso

1. Inicia la aplicación ejecutando el comando `streamlit run app.py`.
